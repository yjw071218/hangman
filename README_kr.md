# 행맨 게임 (C++ 콘솔 애플리케이션)

## 개요
이 행맨 게임은 콘솔 기반의 단어 추측 게임으로, 플레이어는 문자를 제안하여 숨겨진 단어를 맞추어야 합니다. 게임은 **동물**, **과일**, **기술** 등 다양한 카테고리를 제공하며, 각 단어는 난이도(쉬움, 보통, 어려움, 익스트림)에 따라 분류됩니다. 플레이어는 단어 난이도에 따라 점수를 획득하고, 연속으로 맞추는 경우 추가 보너스 점수도 받을 수 있습니다. 또한 게임은 플레이어의 점수를 기록하여 랭킹 시스템에서 비교할 수 있도록 지원합니다.

이 게임은 **C++**로 작성되었으며, **Windows API**를 사용하여 콘솔 색상 및 화면 버퍼링을 처리합니다. 게임은 터미널 창에서 실행되며, 텍스트 기반의 그래픽을 통해 레트로한 게임 플레이 경험을 제공합니다.

## 기능
- **다양한 카테고리**:
    - **동물**, **과일**, **기술** 등 여러 카테고리 중에서 선택할 수 있습니다. 각 카테고리는 다양한 난이도의 단어를 포함하고 있습니다.
    - 카테고리는 쉬움, 보통, 어려움, 익스트림 난이도로 구분된 단어들을 포함하며, 플레이어는 난이도를 선택하여 도전할 수 있습니다.

- **난이도 설정**:
    - 쉬운 단어는 1000점을, 익스트림 단어는 4000점을 제공합니다.
    - 각 난이도는 단어의 복잡도뿐만 아니라 틀릴 수 있는 기회의 수에도 영향을 미칩니다.

- **보너스 스트릭**:
    - 연속으로 맞춘 글자마다 추가 보너스 점수(한 번에 100점)가 주어집니다.
    - 이는 무작위로 추측하는 대신 전략적인 플레이를 장려합니다.

- **랭킹 시스템**:
    - 플레이어가 게임을 마칠 때마다 점수가 저장됩니다. 점수는 **학번**, **플레이어 이름**, **점수**와 함께 기록되어 여러 플레이어가 경쟁할 수 있습니다.

- **스킵 기회**:
    - 플레이어는 게임 중 최대 3번까지 어려운 단어를 스킵할 수 있습니다. 스킵을 현명하게 사용하면 높은 점수를 얻는 데 도움이 됩니다.

- **점수 누적**:
    - 플레이어가 단어를 맞추는 한, 점수는 다른 카테고리에서도 누적됩니다. 이는 장기적인 플레이와 일관성을 보상하는 시스템입니다.

- **타이머**:
    - 각 단어를 맞추기 위해 **30초의 시간**이 주어지며, 시간이 지나면 목숨 하나를 잃게 됩니다.

- **사용자 정의 색상 및 애니메이션**:
    - 각 난이도에 따라 색상이 다릅니다: 쉬움은 초록색, 보통은 노란색, 어려움은 빨간색, 익스트림은 보라색으로 표시됩니다.
    - **진행 바**는 각 추측에 남은 시간을 시각적으로 보여주며, 시간이 부족할 때는 타이머가 흔들리는 애니메이션이 추가됩니다.

## 게임 방법

1. **게임 시작**:
    - 컴파일된 바이너리 또는 스크립트를 실행하여 게임을 시작합니다. 메뉴에서 게임을 실행, 규칙 보기, 랭킹 보기 옵션을 선택할 수 있습니다.

2. **플레이어 정보 입력**:
    - 플레이어는 **학번**(5자리 숫자)과 **이름**을 입력해야 합니다. 이 정보는 랭킹 시스템에서 플레이어의 성적을 추적하는 데 사용됩니다.

3. **카테고리 선택**:
    - 방향키를 사용하여 카테고리를 탐색하고 선택합니다. 각 카테고리는 다양한 난이도의 단어를 포함하고 있습니다.

4. **단어 추측**:
    - 게임은 "코끼리" 같은 단어에 대해 “긴 코를 가진 큰 동물” 같은 힌트를 제공합니다.
    - 문자를 입력하여 추측합니다. 맞으면 단어가 공개되며, 틀리면 기회를 잃게 됩니다.
    - **Tab** 키를 눌러 단어를 건너뛸 수 있으며, 스킵은 3번까지 가능합니다.

5. **점수**:
    - 단어 난이도에 따라 점수가 주어집니다:
      - 쉬움: 1000점
      - 보통: 1500점
      - 어려움: 2000점
      - 익스트림: 4000점
    - **보너스 점수**: 연속으로 맞출 때마다 100점씩 추가됩니다.

6. **게임 종료**:
    - 6번의 틀린 추측을 하면 게임이 종료됩니다. 또한 타이머가 끝나면 게임이 끝나게 됩니다.
    - 단어를 모두 맞추면 카테고리 선택으로 돌아가거나 게임을 계속 진행할 수 있습니다.

7. **점수 저장**:
    - 점수는 자동으로 `ranking.txt` 파일에 저장되며, 이름, 학번, 게임을 진행한 시간과 함께 기록됩니다.

## 게임 로직

- **단어 선택**:
    - 단어는 선택한 카테고리에서 무작위로 선택됩니다. 플레이어가 맞추거나 스킵한 단어는 제거됩니다.
  
- **행맨 그림**:
    - 틀린 추측을 할 때마다 행맨 그림이 점점 그려집니다. 6번의 잘못된 추측 후 플레이어는 패배하며, "게임 오버" 메시지와 함께 최종 그림이 표시됩니다.
  
- **타이머 애니메이션**:
    - 진행 바와 타이머가 남은 시간을 화면에 보여줍니다. 시간이 10초 이하로 남으면 타이머가 흔들리며 긴장감을 높입니다.

- **점수 계산**:
    - 점수는 단어의 난이도에 따라 부여됩니다. 플레이어가 연속으로 맞춘 경우 추가 보너스 점수가 주어집니다.

## 메뉴 옵션

- **게임 시작**:
    - 플레이어 정보를 입력하고 카테고리를 선택하여 게임을 시작합니다.
  
- **규칙 보기**:
    - 게임 규칙을 보여주며, 점수 계산 방법을 설명합니다.
  
- **랭킹 보기**:
    - 이전 플레이어들의 점수를 기반으로 한 리더보드를 표시합니다.

- **게임 종료**:
    - 게임을 종료하고 터미널로 돌아갑니다.

## 문제 해결

### 일반적인 문제

1. **랭킹 시스템이 저장되지 않음**:
    - 게임이 `ranking.txt` 파일에 쓰기 권한이 있는지 확인하세요.
    - 게임 실행 중에 다른 프로그램에서 `ranking.txt` 파일이 열려 있지 않은지 확인하세요.

2. **입력 문제 (백스페이스가 작동하지 않음)**:
    - 일부 터미널에서는 백스페이스 키가 예상대로 작동하지 않을 수 있습니다. 게임은 이를 처리하기 위해 백스페이스 입력 시 문자를 삭제하는 기능을 제공합니다.

3. **콘솔 색상이 제대로 표시되지 않음**:
    - 게임은 콘솔에서 색상을 관리하기 위해 Windows API를 사용합니다. 콘솔이 색상을 지원하는지, 게임이 호환되는 환경(예: Windows 명령 프롬프트)에서 실행 중인지 확인하세요.

4. **타이머가 제대로 작동하지 않음**:
    - 타이머는 `std::chrono`와 정확한 시스템 시계를 기반으로 합니다. 타이머가 일관되지 않게 작동하면 시스템 시계 설정을 확인하세요.

## 폴더 구조

- **소스 코드**:
    - 전체 게임 로직은 단일 C++ 파일(`hangman.cpp`)에 포함되어 있습니다. 이 파일에는 메뉴, 게임 로직, 입력 처리 등이 포함되어 있습니다.
  
- **랭킹 파일**:
    - 각 게임이 끝나면 플레이어의 점수는 `ranking.txt` 파일에 저장됩니다. 이 파일은 학번, 플레이어 이름, 점수, 게임을 진행한 시간을 기록합니다.

## 컴파일

다음 명령어를 사용하여 C++ 컴파일러(예: `g++`)로 게임을 컴파일하세요:

```bash
g++ -o hangman hangman.cpp
```

컴파일 후에는 아래 명령어로 게임을 실행할 수 있습니다:

```bash
./hangman
```

실행 파일에 필요한 권한이 있는지 확인하고, `ranking.txt` 파일이 실행 파일과 같은 디렉토리에 있어야 랭킹 시스템이 올바르게 작동합니다.

## 추가 참고 사항

- 이 게임은 **Windows**용으로 설계되었으며, 콘솔 색상 관리 및 화면 처리를 위해 Windows API를 사용합니다. 비 Windows 플랫폼에서는 일부 기능이 제대로 작동하지 않을 수 있습니다.
- 이 게임은 `std::chrono`를 사용하여 시간 관리를 수행하므로 시스템 시계가 정확하게 설정되어 있는지 확인하세요.
