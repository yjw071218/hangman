# 행맨 게임 (C++ 콘솔 애플리케이션)

## 개요
이 행맨 게임은 플레이어가 단어를 추측해 숨겨진 단어를 찾아가는 콘솔 기반 게임입니다. 동물, 과일, 기술 등 다양한 카테고리에서 단어를 선택할 수 있으며, 각 단어는 난이도에 따라 점수를 획득할 수 있습니다. 게임은 플레이어의 진행 상황을 추적하고, 건너뛰기 기회 및 순위 시스템을 제공합니다. 연속으로 올바른 문자를 맞추면 추가 점수를 얻을 수 있어 전략적인 플레이를 장려합니다.

이 게임은 **C++**로 작성되었으며, 콘솔에서 색상 변경과 화면 버퍼링을 위한 **Windows API 함수**를 사용합니다. 텍스트 기반의 그래픽을 통해 복고풍의 게임 경험을 제공합니다.

## 주요 기능
- **다양한 카테고리**:
    - **동물**, **과일**, **기술** 등의 카테고리에서 단어를 선택할 수 있습니다. 각 카테고리는 다양한 난이도의 단어로 구성되어 있으며, 플레이어는 원하는 카테고리를 자유롭게 선택할 수 있습니다.
    - 카테고리는 난이도에 따라 Easy, Medium, Hard, Extreme로 구분되며, 각 난이도는 랜덤으로 선택됩니다.

- **난이도별 점수**:
    - Easy 단어는 1000점, Extreme 단어는 4000점을 제공합니다.
    - 난이도는 단어의 어려움과 잃을 수 있는 기회에 영향을 미칩니다.

- **보너스 스트릭**:
    - 연속으로 올바른 문자를 맞출 때마다 추가 점수를 얻습니다 (스트릭 당 100점).
    - 신중하고 전략적인 추측을 장려합니다.

- **순위 시스템**:
    - 각 게임이 끝난 후 플레이어의 점수가 저장되며, 다른 플레이어와 점수를 비교할 수 있습니다.
    - 학생 번호, 플레이어 이름, 점수가 저장되며, 동일 컴퓨터에서 여러 플레이어가 경쟁할 수 있습니다.

- **건너뛰기 기회**:
    - 각 게임에서 3번의 건너뛰기 기회를 제공하여 어려운 단어를 건너뛸 수 있습니다.

- **점수 유지**:
    - 단어를 올바르게 추측할 때마다 점수가 유지되며, 카테고리 전환 시에도 이어집니다.

- **타이머**:
    - 각 추측마다 **30초** 타이머가 있으며, 시간을 초과하면 기회를 잃습니다.

- **색상 및 애니메이션 사용자 정의**:
    - 난이도에 따라 색상이 다릅니다: Easy는 초록색, Medium은 노란색, Hard는 빨간색, Extreme은 보라색으로 표시됩니다. 또한 **진행 바**가 남은 시간을 표시합니다.
    - 오답을 맞출 때마다 행맨 그림이 업데이트되고, 시간이 줄어들수록 타이머가 흔들리며 압박감을 줍니다.

- **이중 버퍼링**:
    - 이중 버퍼링을 사용하여 깜박임 없이 부드러운 콘솔 업데이트를 제공합니다.

- **로딩 애니메이션**:
    - 로딩 스피너, 점수 애니메이션, 환영 메시지 등 다양한 애니메이션을 통해 사용자 경험을 향상시킵니다.

## 플레이 방법

1. **게임 시작**: 
    - 프로그램을 실행하면 게임 플레이, 규칙 보기, 순위 확인 등의 옵션을 선택할 수 있습니다.

2. **플레이어 정보 입력**: 
    - 학생 번호(5자리 숫자)와 이름을 입력하면 순위 시스템에서 추적됩니다.

3. **카테고리 선택**: 
    - 화살표 키를 사용하여 카테고리를 선택하고, 원하는 카테고리를 선택한 후 단어를 맞추기 시작합니다.

4. **단어 추측**: 
    - 제공된 힌트를 보고 단어를 추측하세요.
    - 문자를 추측하여 맞으면 단어에 표시되고, 틀리면 기회를 잃습니다.
    - **Tab** 키를 눌러 단어를 건너뛸 수 있으며, 3번의 건너뛰기 기회가 주어집니다.

5. **점수 계산**:
    - 단어 난이도에 따라 점수를 얻습니다:
      - Easy: 1000점
      - Medium: 1500점
      - Hard: 2000점
      - Extreme: 4000점
    - **보너스 점수**: 연속으로 맞춘 경우 스트릭 당 100점이 추가됩니다.

6. **게임 종료**:
    - 6번의 오답을 맞추거나 타이머가 종료되면 게임이 끝납니다.
    - 단어를 맞췄다면 계속해서 다음 단어를 추측하거나 새로운 카테고리를 선택할 수 있습니다.
    - 게임이 끝나면 점수가 저장되고 순위 시스템에 기록됩니다.

7. **점수 저장**:
    - 게임 종료 후 `ranking.txt` 파일에 점수, 이름, 학생 번호 및 타임스탬프가 자동으로 저장됩니다.

## 게임 로직

### 단어 선택
- 단어는 선택한 카테고리에서 무작위로 선택되며, 한 번 맞추거나 건너뛴 단어는 다시 나타나지 않습니다. 플레이어는 카테고리 내의 단어를 모두 맞추거나 기회를 모두 소진할 때까지 게임을 계속 진행할 수 있습니다.

### 행맨 그림
- 오답을 맞출 때마다 행맨 그림이 순차적으로 그려집니다. 플레이어는 6번의 기회를 제공받으며, 오답을 맞출 때마다 기회가 하나씩 줄어듭니다. 6번의 오답을 맞추면 행맨 그림이 완성되고, 게임이 종료됩니다.

### 점수 시스템
- 플레이어는 단어 난이도에 따라 점수를 얻습니다:
  - Easy: 1000점
  - Medium: 1500점
  - Hard: 2000점
  - Extreme: 4000점
- 올바른 문자를 연속으로 맞출 때마다 스트릭 당 100점의 추가 점수를 얻습니다.
- 플레이어는 여러 카테고리를 넘나들며 점수를 축적할 수 있으며, 게임이 끝나면 최종 점수가 순위 시스템에 저장됩니다.
- 생명을 모두 유지한 상태에서 카테고리를 완료하면 해당 카테고리에서 얻은 점수의 10%가 보너스로 추가됩니다.

### 타이머 시스템
- 각 추측은 **30초** 내에 이루어져야 하며, 화면에 진행 바와 타이머가 표시됩니다. 시간이 다 되면 기회를 하나 잃게 됩니다.
- 남은 시간이 **10초 이하**로 떨어지면 타이머가 흔들리며 시각적인 경고를 제공합니다.

### 건너뛰기 기회
- 플레이어는 게임당 최대 3번의 건너뛰기 기회를 제공합니다. 단어를 건너뛰어도 기회는 줄어들지 않지만, 사용할 수 있는 건너뛰기 횟수가 줄어듭니다.

### 순위 및 점수 저장
- 게임이 끝나면 플레이어의 **학생 번호**, **이름**, **점수**가 `ranking.txt` 파일에 저장됩니다. 플레이어는 이전 게임 참가자들과 점수를 비교할 수 있으며, 점수는 점수와 타임스탬프에 따라 정렬됩니다.

## 메뉴 옵션

1. **게임 시작**: 
    - 학생 번호와 이름을 입력한 후 카테고리를 선택하고 단어를 추측하는 게임을 시작하세요.
  
2. **규칙 보기**: 
    - 게임 규칙, 점수 계산 방식 및 게임 플레이 설명을 표시합니다.

3. **순위 확인**: 
    - 이전 플레이어들의 점수가 기록된 리더보드를 표시하며, 높은 점수 순으로 정렬됩니다.

4. **종료**: 
    - 게임을 종료하고 터미널로 돌아갑니다.

## 문제 해결

### 자주 발생하는 문제

1. **순위 시스템 저장 실패**:
    - 게임이 `ranking.txt` 파일을 저장할 수 있는 폴더에 쓰기 권한이 있는지 확인하세요.
    - 게임 실행 중에 다른 프로그램에서 파일이 열려있지 않은지 확인하세요.

2. **입력 문제 (백스페이스 미작동)**:
    - 일부 터미널에서는 백스페이스 키가 정상적으로 작동하지 않을 수 있습니다. 게임은 이를 처리하여 백스페이스를 누르면 입력된 문자를 삭제할 수 있습니다.

3. **콘솔 색상 표시 문제**:
    - 게임

은 Windows API를 사용하여 콘솔 색상을 처리합니다. 터미널이 색상을 지원하는지, Windows의 명령 프롬프트에서 게임이 실행되고 있는지 확인하세요.

4. **타이머 작동 문제**:
    - 타이머는 `std::chrono` 및 시스템의 정확한 시간 관리 기능에 의존합니다. 타이머가 일관되지 않다면 시스템 시간 설정을 확인하세요.

5. **행맨 그림이 표시되지 않음**:
    - 콘솔 창이 행맨 그림을 표시하기에 충분한 크기인지 확인하세요. 필요하다면 창 크기를 조정하세요.

## 폴더 구조

게임은 다음과 같은 폴더 구조를 기대합니다:

```
/hangman
│
├── /bin           # 컴파일된 실행 파일
│   └── hangman.exe
│
├── /data          # 게임 데이터 파일
│   └── categories.txt    # 단어, 힌트 및 난이도 데이터
│   └── ranking.txt       # 플레이어 순위 및 점수 데이터
│
├── /src           # 소스 코드
│   └── hangman.cpp
│
└── README.md      # 게임 문서 파일
```

- 게임이 제대로 작동하려면 `categories.txt`(단어, 힌트, 난이도 정보 포함)와 `ranking.txt`(플레이어 점수 저장) 파일이 `data` 폴더에 있어야 합니다.

## 컴파일

다음 명령어를 사용하여 C++ 컴파일러(예: `g++`)로 게임을 컴파일하세요:

```bash
g++ -o hangman hangman.cpp
```

컴파일 후에는 아래 명령어로 게임을 실행할 수 있습니다:

```bash
./hangman
```

## 참고 사항

- **호환성**: 
    - 이 게임은 콘솔 색상 관리 및 화면 처리를 위해 Windows API를 사용하므로 **Windows**에서 동작하도록 설계되었습니다. 다른 운영 체제에서는 API 호출을 조정해야 할 수 있습니다.
  
- **시스템 요구 사항**: 
    - 게임은 시간 관리를 위해 `std::chrono`를 사용하므로, 시스템 시간이 올바르게 설정되어 있어야 타이머가 정상적으로 작동합니다.
  
- **파일 경로**: 
    - 순위 시스템이 정상적으로 작동하려면 `ranking.txt` 파일이 실행 파일과 동일한 디렉토리에 있어야 합니다.
